# Trained Model

The trained CNN model file is not included in this repository due to GitHub
file size limitations.

## Model Details
- Architecture: Convolutional Neural Network (CNN)
- Input Size: 64 × 64 RGB images
- Output Classes: 71 (Letters, Numbers, and Gestures)
- Framework: TensorFlow / Keras
